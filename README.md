# RAMPG
делаем простую консольную 3д РПГ, про то как безимянный герой должен победить сема альтмана вместе с CEO нвидии, по пути получая оперативку? За больше оперативки можно увеличить рендеринг или получить новую способность(условно взрывы) в С игра будет написана чтобы потреблять минимум байт оперативки


откройте папку проекта и напишите 
```make``` для компиляции
и
```make clean``` для очистки приложения

Стилистический гайд появится позже
